# ProyectoGitDAM
   
   Programa en Java que muestra un mensaje por consola utilizando Git y GitHub.
   
   ## Descripción
   
   Este proyecto es una introducción al uso de Git como sistema de control de versiones y GitHub como plataforma de alojamiento de repositorios.
   
   ## Lenguaje utilizado
   
   - Java
   
   ## Estructura del proyecto
   
   - Paquete: `es.dam.git`
   - Clase principal: `HolaGit.java`
   
   ## Ejecución
   
   1. Abrir el proyecto en Eclipse
   2. Ejecutar la clase `HolaGit` desde Eclipse
   3. El programa mostrará un mensaje por consola
   
   ## Tecnologías
   
   - Java
   - Git
   - GitHub
   - Eclipse IDE
